# ABX!
## Motivation
I've always found games fascinating, and the idea of making them even more so. This was my first attempt at making a game! It started as a small Galaga remake in C to learn fundamental game development. However, the more I worked on it, the more ideas started coming, and it evolved into a tactical Galaga-like shooter with procedurally generated seeded levels and a Path of Exile-style player leveling tree.

In this process, I focused on data-oriented design, entity component systems, and DOOM-like architecture. The state of this repo represents the last working build before I switched to rewriting in Odin. The reason for the switch is that every time I wanted to iterate or build something new, C would get in the way of making quick progress. I really enjoy the batteries-included philosophy of Odin and the fact that it gets out of the way to let you build.

**This repository is archived** - active development continues in [[ABX_odin](https://github.com/markwilliamsjr/ABX_odin)].
